# 🔥 AGNI --- Royal Indian Fine Dining Website

This project is a premium restaurant website UI built from scratch using
HTML, CSS, and vanilla JavaScript. It focuses on modern frontend design
practices, responsive layout engineering, cultural branding elements,
and smooth user interface interactions.

The goal of this project is to demonstrate advanced frontend development
skills through a visually rich restaurant website that simulates a
luxury Indian dining experience.

------------------------------------------------------------------------

## 🚀 Features

-   **Responsive UI Design**: Fully adaptable layout across desktop,
    tablet, and mobile devices.\
-   **Sticky Navigation Bar**: Dynamic navigation behavior with scroll
    effects.\
-   **Interactive Menu Tabs**: Dynamic switching between food
    categories.\
-   **Animated UI Components**: Hover effects, scroll reveal animations,
    and transitions.\
-   **Custom Cursor Interaction**: Enhanced visual engagement through
    cursor animation.\
-   **Gallery Showcase**: Structured grid gallery with hover overlays.\
-   **Reservation Form UI**: Styled form layout for booking simulation.\
-   **Cultural Design Elements**: Typography, color palette, and
    decorative motifs inspired by Indian fine dining themes.

------------------------------------------------------------------------

## 🧱 System Overview

This is a frontend-only single-page website.\
There is no backend server or database integration.

User Browser → HTML Structure → CSS Styling + Animations → JavaScript
Interactions → Rendered Restaurant UI

------------------------------------------------------------------------

## 🛠 Tech Stack

### Frontend
